# img2cli v0.3.0 Tauri GUI & Text Injection Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a premium System Tray-resident GUI desktop application for img2cli using Tauri v2, providing visual SSH/Local host profile management and clipboard-free Markdown link pasting via direct OS text injection or quick clipboard swap.

**Architecture:**
*   **Tauri v2 Core**: Native Rust backend managing the tray menu, startup autostart, global shortcuts, and file upload daemon threads.
*   **Tauri Commands (IPC)**: Provides commands to read/write `config.toml` and test SSH connections.
*   **Tauri WebView Frontend**: Vue 3 + Vite + Tailwind CSS single-page settings dashboard with glassmorphism aesthetics.
*   **Direct Injection & Swapper**: Rust API utilizing Win32 SendInput or a quick clipboard swap-and-restore fallback to paste Markdown links instantly without corrupting the clipboard image.

**Architecture Diagram:**

```mermaid
graph TD
    subgraph Frontend (Vite + Vue 3 + Tailwind)
        index[index.html] --> main_js[src/main.js]
        main_js --> app_vue[src/App.vue]
        style_css[src/index.css] --> app_vue
    end

    subgraph Backend (Tauri Rust Core)
        main_rs[src-tauri/src/main.rs] --> daemon[src-tauri/src/daemon.rs]
        main_rs --> injector[src-tauri/src/injector.rs]
        main_rs --> config_rs[src-tauri/src/config.rs]
    end

    app_vue <-->|IPC Tauri Invokes| main_rs
    daemon -->|Uploads Image| SSH[Remote Server via SCP]
    daemon -->|Invokes Paste| injector
    injector -->|direct: Sends OS Input / swap: Quick Swap| Terminal[Target Focus Window]
```

**Tech Stack:**
*   Rust 1.75+ & Cargo
*   Tauri v2
*   Tauri v2 plugins: Autostart, Global Shortcut, Shell
*   NodeJS & npm
*   Vue 3, Vite, Tailwind CSS v3.4

## Global Constraints
*   Minimum Rust version: 1.75
*   Tauri version: v2.0
*   Config file: `config.toml` (persistent path: user config directory)
*   Tray resident behavior: Close window must hide window, keeping daemon active.

  use std::path::PathBuf;

  #[derive(Debug, Clone, Serialize, Deserialize)]
  pub struct SshConfig {
      pub enabled: bool,
      pub host: String,
      pub port: Option<u16>,
      pub username: Option<String>,
      pub remote_dir: String,
      pub match_pattern: Option<String>,
  }

  #[derive(Debug, Clone, Serialize, Deserialize)]
  pub struct TargetConfig {
      pub enabled: bool,
      pub r#type: String, // "ssh" or "local"
      pub match_pattern: String,
      pub host: Option<String>,
      pub port: Option<u16>,
      pub username: Option<String>,
      pub remote_dir: Option<String>,
      pub local_dir: Option<String>,
  }

  #[derive(Debug, Clone, Serialize, Deserialize)]
  pub struct AppConfig {
      pub output_format: String,
      pub compress_quality: u8,
      pub max_dimension: Option<u32>,
      pub wrap_single_quotes: bool,
      pub launch_on_boot: bool,
      pub enable_notifications: bool,
      pub global_hotkey: String,
      pub upload_strategy: String, // "eager" or "lazy"
      pub injection_mode: String,  // "direct" or "swap"
      pub clean_keep_days: u32,
      pub ssh: Option<SshConfig>,
      pub targets: Option<Vec<TargetConfig>>,
  }

  pub fn get_config_path() -> PathBuf {
      tauri::api::path::config_dir()
          .unwrap_or_else(|| std::env::current_dir().unwrap())
          .join("img2cli")
          .join("config.toml")
  }
